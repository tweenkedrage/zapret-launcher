# Zapret Launcher
`Обход ограничений РКН, ускорение Telegram/YouTube и снижение задержек в играх`

***Установка:***
1. [Скачивайте любой архив](https://github.com/tweenkedrage/zapret-launcher/releases/latest) (`.rar`, `.zip`) и распаковывайте в любое место 2 файла: `Zapret_Launcher.exe` и `zapret_resources.zip` (Главное чтобы они были в одном месте и не в отдельных папках);
2. Запускайте **Zapret_Launcher.exe** от **имени администратора**;
3. Выбирайте любой режим использования;
4. Пользуйтесь.

***Telegram Proxy:***
1. Выбирайте режим **TG Proxy**;
2. Переходите в настройки **Telegram**;
3. **Продвинутые настройки**;
4. **Тип соединения**;
5. Использовать собственное прокси (**SOCKS5**, Хост: **127.0.0.1**, Порт: **1080**).

***ByeDPI оптимизатор:***
1. Включайте только если тормозит интернет или стратегии **Zapret** не помогают;
2. Полезен для **YouTube** и **онлайн-игр**.

***zapret_resources.zip:***
1. Это архив со всеми файлами оригинального **Zapret`a**, которые необходимы для работы лаунчера;
2. При первом запуске лаунчер распаковывает **zapret_resources.zip** в `AppData/Local/ZapretLauncher/zapret_core`;

***НЕ УДАЛЯЙТЕ zapret_resources.zip, если:***
1. Папка **zapret_core** отсутствует или повреждена;
2. Вы хотите сохранить возможность переустановки без скачивания;
3. Вы хотите поделиться программой (Архив **zapret_resources.zip** должен быть рядом с .exe лаунчера).

***Антивирусы:***
- Некоторые антивирусы могут реагировать на программу из-за использования компонента **WinDivert**. Это **НОРМАЛЬНО**;
- **WinDivert** — это легальный драйвер с открытым исходным кодом, используемый для фильтрации сетевых пакетов.

***Если антивирус ругается:***
- Добавьте папку с лаунчером в **Исключения**;
- Или скомпилируйте программу из исходников или временно отключите антивирус при запуске.

`Zapret и ByeDPI работают на разных уровнях и в большинстве случаев не конфликтуют.`
