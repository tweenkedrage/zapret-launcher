> [!WARNING]
>
> ### АНТИВИРУСЫ
> Антивирусы могут жаловаться на файл **WinDivert**.
> **WinDivert** - это инструмент для **перехвата** и **фильтрации трафика**, необходимый для работы **zapret**.
>
> **WinDivert** работает на низком уровне системы, перехватывая сетевые пакеты до того, как они достигают приложений.
> Такое поведение типично для вредоносного ПО (например, для **снифферов** или **троянов**),
> поэтому антивирусы могут ложно определять его как угрозу.
>
> ## Антивирус жалуется?
> ### 1. Windows Defender:
> - Параметры → Конфиденциальность и безопасность → Безопасность **Windows**;
> - Защита от вирусов и угроз → Управление настройками;
> - Исключения → Добавить папку с **лаунчером**.
> ### 2. Kaspersky, Dr.Web и др:
> - Откройте настройки **антивируса**;
> - Найдите раздел **Исключения** или **Доверенные приложения**;
> - Добавьте папку `%APPDATA%\ZapretLauncher\` в **белый список**.

> [!NOTE]
> Если вы скачали лаунчер с [официального репозитория](https://github.com/tweenkedrage/zapret-launcher) **GitHub**,
> можете быть **уверены** в **безопасности** лаунчера.
> При необходимости проверьте файлы через [**VirusTotal**](https://www.virustotal.com/gui/home/upload).

## Основные возможности

## 🚀 Локальный Telegram прокси:
- Ускоряет работу **Telegram** без необходимости использовать **сторонние прокси-серверы**
- **Автоматическое переключение** на рабочие домены **при блокировках**
- Умный пул **WebSocket-соединений** с **балансировкой** нагрузки
- **Автоматическая генерация** секретного ключа
- Поддержка **Fake TLS** для маскировки трафика

## 🌐 Zapret
- Готовый набор стратегий **для обхода DPI**
- Встроенный редактор списков (**домены**, **IP-адреса**)
- Поддержка пользовательских списков (`list-custom.txt`)
- Фильтры: **Game Filter**, **IPSet Filter**

## 📊 Мониторинг
- Трафик по процессам: таблица с разделением на **VPN**/**прямой трафик**
- Скорость соединения (**⬆ отдача** / **⬇ загрузка**) в **реальном** времени
- Статистика сессии: **время работы**, **общий трафик**, **RTT** (ping)
- Логирование событий **лаунчера**

## 🔧 Дополнительные инструменты
- Разблокировка **SoundCloud** (автоматическое добавление доменов)
- Разблокировка **Meta** (автоматическое добавление доменов)
- Инструкция по ручной разблокировке **Telegram Web** (через **hosts**)
- Инструкция по восстановлению доступа к **GitHub**
- Копирование необходимых **доменов**/**IP** в буфер обмена

## 🎮 Режимы работы
### Стандартный:
- Обход блокировок через **Zapret**
### Telegram Proxy:
- Ускорение **Telegram** через встроенный прокси
### Совместный
- **Zapret** & **Telegram Proxy**

## ⚙️ Настройки
- **Автозапуск** вместе с **Windows**
- Настройка скорости **обновления интерфейса**
- 2 темы оформления: **Default** (Black), **Pink**
- Поддержка 3 языков: **Русский**, **Українська**, **English**

## 🖥️ Системные требования
- ОС: Windows **10**/**11** (64-bit)
- Права администратора (для работы **Zapret**)
