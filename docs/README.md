# Zapret Launcher 
Универсальный лаунчер и менеджер для инструментов обхода блокировок

## Основные возможности

#### Локальный прокси для Telegram:
- Поддержка **Cloudflare fallback** - автоматическое переключение на рабочие домены при блокировках;
- Ускоряет работу **Telegram** без необходимости использовать сторонние **прокси**-сервера;
- Умный пул **WebSocket** соединений с балансировкой нагрузки;
- Автоматическая генерация секрет-ключа.

#### Zapret (DPI обход):
- Выбор стратегий обхода из предустановленного набора;
- Редактирование списков.

#### Мониторинг:
- **Трафик по процессам**: детальная таблица с разделением на **VPN**/**прямой трафик**, **скорость**, количество **соединений**;
- **Статистика сессии**: **время работы**, **трафик** (`⬆/⬇`), **текущая** скорость, **RTT** задержка.

#### Дополнительные инструменты:
- Добавление доменов для **SoundCloud**;
- Добавление доменов для **Meta** (`FaceBook`, `Instagram`);
- Ручное добавление доменов для **Telegram Web**;
- Ручное добавление доменов для **GitHub**.

#### Режимы работы:
- **Стандартный** - обход блокировок через **Zapret**;
- **Telegram Proxy** - ускорение **Telegram** через встроенный прокси;
- **Совместный** - **Zapret** и **Telegram Proxy** одновременно.

## Требования:
- Windows **10** / **11** (64-bit)

> [!WARNING]
>
> ### АНТИВИРУСЫ
> Антивирусы могут жаловаться на файл **WinDivert**.
> **WinDivert** - это инструмент для **перехвата** и **фильтрации трафика**, необходимый для работы **zapret**.
>
> **WinDivert** работает на низком уровне системы, перехватывая сетевые пакеты до того, как они достигают приложений.
> Такое поведение типично для вредоносного ПО (например, для **снифферов** или **троянов**),
> поэтому антивирусы могут ложно определять его как угрозу.
>
> ## Антивирус жалуется?
> ### 1. Windows Defender:
> - Параметры → Конфиденциальность и безопасность → Безопасность **Windows**;
> - Защита от вирусов и угроз → Управление настройками;
> - Исключения → Добавить папку с **лаунчером**.
> ### 2. Kaspersky, Dr.Web и др:
> - Откройте настройки **антивируса**;
> - Найдите раздел **Исключения** или **Доверенные приложения**;
> - Добавьте папку `%APPDATA%\ZapretLauncher\` в **белый список**.
