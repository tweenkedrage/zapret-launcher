## Основные возможности

## 🚀 Локальный Telegram прокси:
- Ускоряет работу **Telegram** без необходимости использовать **сторонние прокси-серверы**
- **Автоматическое переключение** на рабочие домены **при блокировках**
- Умный пул **WebSocket-соединений** с **балансировкой** нагрузки
- **Автоматическая генерация** секретного ключа
- Поддержка **Fake TLS** для маскировки трафика

## 🌐 Zapret
- Готовый набор стратегий **для обхода DPI**
- Встроенный редактор списков (**домены**, **IP-адреса**)
- Поддержка пользовательских списков (`list-custom.txt`, `ipset-white-user.txt`)
- Фильтры: **Game Filter**, **IPSet Filter**

## 📊 Мониторинг
- Трафик по процессам: таблица с разделением на **VPN**/**прямой трафик**
- Скорость соединения (**⬆ отдача** / **⬇ загрузка**) в **реальном** времени
- Статистика сессии: **время работы**, **общий трафик**, **RTT** (ping)

## 🔧 Дополнительные инструменты
- Разблокировка **SoundCloud** (автоматическое добавление доменов)
- Разблокировка **Meta** (автоматическое добавление доменов)
- Инструкция по ручной разблокировке **Telegram Web** (через **hosts**)
- Инструкция по восстановлению доступа к **GitHub**

## 🎮 Режимы работы
| Режим | Описание |
|--------|-------------|
| **Стандартный** | Обход блокировок через zapret |
| **Telegram Proxy** | Ускорение Telegram через встроенный прокси |
| **Совместный** | Zapret & Telegram Proxy |

## ⚙️ Настройки
- **Автозапуск** вместе с **Windows**
- Настройка скорости **обновления интерфейса**
- 2 темы оформления: **Default** (Black), **Pink**
- Поддержка 2-х языков: **Русский** и **English**
