# Zapret Launcher
`Обход ограничений РКН и ускорение работы Telegram в РФ`

***Установка:***
1. [Скачайте любой архив](https://github.com/tweenkedrage/zapret-launcher/releases/latest) (`.rar`, `.zip`) и распакуйте в любое место 2 файла: `Zapret_Launcher.exe` и `zapret_resources.zip` (Главное чтобы они были в одном месте и не в отдельных папках);
2. Запустите **Zapret_Launcher.exe** от **имени администратора**;
3. Выберите любой режим использования;
4. Пользуйтесь.

***Нагрузка на ЦП/ОЗУ:***
- Уменьшите скорость **обновления интерфейса**, страница **Трафик** не работает если она свернута или же лаунчер **закрыт**.

***Telegram Proxy:***
1. Выберите режим **TG Proxy**;
2. Переходите в настройки **Telegram**;
3. **Продвинутые настройки**;
4. **Тип соединения**;
5. Использовать собственное прокси (**SOCKS5**, Хост: **127.0.0.1**, Порт: **1080**).

***ByeDPI:***
1. Включайте только если тормозит интернет или стратегии **Zapret** не помогают;
2. Полезен для **YouTube** и **онлайн-игр**.

***zapret_resources.zip:***
1. Это архив со всеми файлами оригинального **Zapret`a**, которые необходимы для работы лаунчера;
2. При первом запуске лаунчер распаковывает **zapret_resources.zip** в `AppData/Local/ZapretLauncher/`.

***НЕ УДАЛЯЙТЕ zapret_resources.zip если:***
1. Папка **zapret_core** отсутствует или повреждена;
2. Вы хотите поделиться программой (Архив **zapret_resources.zip** должен быть рядом с **.exe лаунчера**).

***Антивирусы:***
- Некоторые антивирусы могут реагировать на программу из-за использования компонента **WinDivert**. Это **НОРМАЛЬНО**;
- **WinDivert** - это легальный драйвер с открытым исходным кодом используемый для фильтрации сетевых пакетов.

***Если антивирус ругается:***
- **1 Вариант:** Добавьте папку с лаунчером в **Исключения/Белый список**;
- **2 Вариант:** Cкомпилируйте программу из исходников или временно отключите антивирус при запуске.

***Системные требования:***
- Windows 10 (64-bit) / Windows 11;
- 2gb RAM.
